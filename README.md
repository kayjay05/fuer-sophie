# Für Sophie

Website zur Unterstützung von Sophie, die gegen einen seltenen, aggressiven Hirntumor (AT/RT) kämpft.

## Über die Seite

Eine statische HTML-Seite, gehostet via [GitHub Pages](https://pages.github.com/) unter [fuer-sophie.at](https://fuer-sophie.at).

## Updates veröffentlichen

1. `index.html` bearbeiten
2. `git add index.html`
3. `git commit -m "Update: ..."`
4. `git push`

GitHub Pages aktualisiert die Seite automatisch nach dem Push.
